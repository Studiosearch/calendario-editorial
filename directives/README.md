# Layer 1: Directives

Este diretório contém os Procedimentos Operacionais Padrão (SOPs) escritos em Markdown.

**Objetivo:**
- Definir as metas, entradas, ferramentas/scripts a serem usados, saídas e casos extremos.
- Instruções em linguagem natural, como se fossem dadas a um funcionário de nível intermediário.
- Estas são as instruções lidas pelo Agente de Orquestração (Layer 2).

*Lembre-se: os arquivos de diretrizes são documentos vivos. Devem ser atualizados à medida que o sistema aprende e se aprimora.*
